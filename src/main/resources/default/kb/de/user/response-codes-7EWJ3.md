description: Die folgende Tabelle beschreibt die Antwortcodes, die standardmäßig von Schnittstellen zurückgegeben werden.
parent: SLEOK
priority: 100
permissions: ""
chapter: false
---
| Code | Beschreibung |
| --- | --- |
| 200 | Normalfall: Aufrufparameter und Anfragemethode sind korrekt. Das Ergebnis der Anfrage wird in den Antwortdetails zurückgeliefert. |
| 400 | Fehlerhafte Eingabe oder Aufrufparameter. |
| 401 | Wenn keine Authentifizierung vorliegt, aber benötigt wird. |
| 403 | Fehlende Berechtigung oder falsche Authentifizierung. |
| 404 | Ressource, insbesondere ein angefragter Inhalt, ein Entity, etc. konnte nicht gefunden werden. |
| 405 | Falsche Anfragemethode, beispielsweise GET statt POST. |
| 500 | Unerwarteter serverseitiger Fehler. |
